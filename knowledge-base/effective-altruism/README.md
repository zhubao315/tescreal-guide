# 有效利他主义 (Effective Altruism)

## 概述

**有效利他主义**(Effective Altruism, EA)是一种实践哲学运动，强调运用证据和理性将善的影响力最大化。其核心问题简单而深刻："如何最有效地帮助他人？"有效利他主义在TESCREAL图谱中扮演"伦理框架"角色——它为其他流派的技术愿景提供道德基础，并指导资源(金钱、时间和职业)的配置方向。

## 核心概念

### 有效利他主义的两大支柱

**1. doing good better**: 使用证据和理性找出最有效的方式帮助他人
**2.  earning to give**: 通过高收入职业积累资源用于慈善

### 核心原则

1. **最大化影响力**: 有限的资源应产生最大的积极影响
2. **证据驱动**: 慈善决策应基于严格的效果评估，而非情感或直觉
3. **理性权衡**: 考虑所有后果，包括长期和间接影响
4. **范围关怀**: 受助者的利益不应因地理或时间距离而被打折

### Cause Prioritization

EA社区关注的优先议题：

| Cause Category | 代表议题 | 核心理由 |
|:---|:---|:---|
| **全球健康** | 贫困、传染病、营养不良 | 高效用的低成本干预 |
| **动物福利** | 工厂化养殖 | 大量个体承受痛苦 |
| **长期主义** | AI风险、生物风险 | 存在性规模 |
| **制度改善** | 政策倡导、制度改革 | 杠杆效应 |

## 历史发展

### 思想渊源 (1970-2000年代)

**彼得·辛格 (Peter Singer)**
1972年，哲学家彼得·辛格发表《饥荒、富足与道德》，提出著名的"拯救儿童"思想实验：如果可以无代价地拯救溺水儿童，我们有道德义务这样做。辛格认为，富国对穷国的援助义务应扩展至日常消费选择。

**威廉·麦askell (William MacAskill)**
2009年，牛津大学哲学系研究生威廉·麦askell和托比·奥德(Toby Ord)开始系统发展EA理念。

### 运动形成 (2009-2013年代)

**2009年**: 
- 麦askell、奥德和其他牛津研究生开始定期聚会讨论EA
- 成立非正式"EA牛津小组"

**2011年**:
- 辛格发表TED演讲《你能做的最大善事》，将EA理念大众化
- LessWrong社区开始广泛讨论EA

**2012年**:
- Giving What We Can (GWWV) 在英国成立
- 成员承诺将至少10%收入捐赠给最有效的慈善机构

**2013年**:
- 麦askell、奥德与乔安娜·克鲁格(Joanna Krupp)创立Centre for Effective Altruism (CEA)
- 《Doing Good Better》一书出版

### 机构化与扩展 (2014-2019)

**2014年**:
- 80,000 Hours成立，提供EA导向的职业咨询服务
- Open Philanthropy成立，成为EA资金的最大来源之一

**2017年**:
- EA Global会议开始举办
- EA社区成员突破10,000人

**2019年**:
- FTX未来基金成立，由Sam Bankman-Fried创立
- EA社区资金规模大幅增长

### 危机与转型 (2022-现在)

**2022年11月**: 
- FTX崩盘，SBF被捕
- EA声誉受到严重打击
- 社区开始反思与金钱权力的关系

**2023年至今**:
- 社区重建信任和治理
- 更多关注多元性和包容性
- 长期主义和AI安全成为更核心的议题

## 核心理论框架

### 功利主义基础

EA通常采用**功利主义**作为道德框架：

- 正确行动 = 最大化总体福祉
- 所有有感知能力的存在具有平等的道德分量
- 后果主义评估——动机和过程不如结果重要

### Cause Prioritization原则

**可解决性**: 这个问题能被解决吗？规模如何？
**规模**: 如果解决了，能带来多大改善？
**边际价值**: 个人额外投入能带来多大边际改善？
**被忽视程度**: 多少资源已投入这个问题？

### Giving What We Can标准

Giving What We Can推荐的慈善机构必须满足：

1. 有效验证：能证明其干预有效的证据
2. 高效：每美元能产生最大的积极影响
3. 被忽视：该问题获得的资源不足
4. 透明：愿意接受效果评估

## EA与长期主义的融合

### 长期主义的兴起

2010年代中期，**长期主义**(Longtermism)逐渐成为EA社区的核心议题：

> "在遥远的未来可能存在数万亿有感知能力的存在，我们的道德义务应扩展至他们。" —— William MacAskill

长期主义者认为：
- 未来世代的福祉与当代人同等重要
- 存在性风险是最大的道德问题
- 降低X-risk具有巨大的"杠杆效应"

### X-Risk与EA

托比·奥德在《悬崖》中估计：

- 人类文明在下一个世纪内面临的存在性风险约为 **1/6** (~17%)
- 核武器、生物武器和AI是主要威胁
- 降低X-risk1%价值约等于拯救1亿亿(~10^16)年人类生命

这使AI安全和生物安全成为EA的核心cause。

## EA与TESCREAL

有效利他主义在TESCREAL中扮演"道德基础设施"角色：

- **→ 超人类主义**: EA提供人体增强技术的伦理评估框架
- **→ 反熵主义**: EA评估延寿技术的资源分配优先级
- **← 奇点主义**: EA评估AI开发的道德正当性
- **← 宇宙主义**: EA评估星际殖民的成本效益
- **← 理性主义**: EA是理性主义价值观的具体应用
- **→ 长期主义**: EA的逻辑延伸，更关注未来世代

## 批评与争议

### 主要批评方向

**精英主义**
EA被批评为精英知识分子的自我服务俱乐部——高收入职业和量化分析服务于受过良好教育的西方人群，而非真正帮助边缘化社区。

**量化过度**
将道德价值量化不可避免地扭曲道德判断——某些价值(如爱、正义)无法被量化。

**忽视系统性变革**
EA过于关注个人行动，忽视了政治和经济结构改革的重要性。

**与权力的暧昧关系**
SBF丑闻暴露了EA社区与科技精英金钱的深度联系。

### 内部批评

**价值多元化**
EA社区内部对"什么算作善"存在分歧——功利主义框架是否过于狭隘？

**长期主义争议**
长期主义是否会导致对当下问题的忽视？
