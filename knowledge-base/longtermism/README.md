# 长期主义 (Longtermism)

## 概述

**长期主义**是一种道德立场，主张当代人的道德责任应延伸至遥远的未来，未来的有感知能力的存在与当代人具有同等甚至更大的道德重要性。长期主义在TESCREAL图谱中是有效利他主义的逻辑延伸和扩展——它将EA的功利主义框架应用于宇宙时间尺度，宣称"存在性风险"(X-risk)是当今世界最重要的道德议题。

## 核心概念

### 长期主义的定义

**基本定义**:
> "长期主义认为，积极影响遥远未来存在的能力具有巨大的道德重要性——不亚于、甚至超过对当下人类的积极影响。"

### 长期主义的三种形式

**1. 描述性长期主义**: 未来人口可能远远超过当代人口

**2. 规范性长期主义**: 未来世代的福祉与当代人同等重要

**3. 实践性长期主义**: 最大化长期积极影响是当下的首要道德任务

### 核心命题

1. **未来可能性巨大**: 如果人类文明延续，未来的生命数量可能是当代的数百亿倍
2. **时间中性**: 功利主义不区分现在和未来的同等福祉
3. **存在性风险是首要议题**: 防止文明毁灭是道德优先项
4. **路径依赖**: 当代决策将深刻影响未来数千年的走向

## 历史发展

### 思想渊源 (1970-2000年代)

**未来世代伦理学**
1970年代，哲学家开始系统研究"代际伦理学"问题：

- 约翰·罗尔斯在《正义论》(1971)中讨论了代际正义
- 德里克·帕菲特(Derek Parfit)在《理由与人格》(1984)中提出"非身份问题"
- 布赖恩·巴里在《正义理论》(1973)中主张未来世代具有平等道德地位

**存在性风险研究**
2000年代，存在性风险(X-risk)成为严肃的学术研究对象：

- 尼克·博斯特罗姆2002年发表《存在性风险》
- 2011年，牛津大学成立人类未来研究所(FHI)

### 长期主义形成 (2010年代)

**2015年**:
- 威廉·麦askell发表《长期主义存在性风险是EA最重要的议题》
- 长期主义开始成为EA社区的核心议题

**2017年**:
- 托比·奥德加入牛津大学，系统研究X-risk
- EA社区资金开始大量流向AI安全和生物安全

**2019年**:
- Hilary Greaves和Theron Pummer在牛津大学组织"长期主义与存在性风险"学术会议
- 长期主义进入主流哲学讨论

### 广泛传播 (2022年至今)

**2022年**:
- 麦askell出版《我们欠未来什么》(What We Owe the Future)
- 《时代》杂志封面报道"长期主义的崛起"
- 批评文章大量涌现

**2023-2026年**:
- 长期主义成为TESCREAL讨论的核心词汇
- AI安全社区大规模扩展
- 学术界和政策界的持续争议

## 核心理论框架

### 存在性风险 (Existential Risk)

**定义**:
> 存在性风险是威胁人类文明永久毁灭或不可逆转衰落的风险——即永远无法恢复的灾难性损失。

**X-risk vs. 生存性风险**:
- **生存性风险(S-risk)**: 导致大量痛苦的风险
- **存在性风险(X-risk)**: 导致文明终结的风险

### X-Risk量化

托比·奥德在《悬崖》中提供的估计：

| 风险来源 | 21世纪X-risk估计 |
|:---|:---|
| 核战争 | ~1% |
| 气候灾难 | <1% |
| 自然流行病 | <1% |
| 工程流行病 | ~1% |
| AI灾难 | ~5-10% |
| 未知风险 | ~1% |
| **总计** | **~1/6 (~17%)** |

### 降低X-Risk的价值

博斯特罗姆的计算：
- 如果X-risk降低了0.01%，拯救了约1亿条生命
- 1亿年未来人类生命的期望值 = 10^16 感知-年
- 降低X-risk1%的道德价值 = 10^14 至 10^17 生命-年

### "代际窗口"概念

麦askell提出的关键概念：

> "我们可能正处于一个特殊的'代际窗口'——一个当代选择将极大影响遥远未来的机会之窗。"

理由：
- 技术能力快速增长
- 制度尚未固化
- 存在性风险尚未实现

## X-Risk的主要来源

### 已确认的X-Risk

**1. 人工智能灾难**
- 超级智能AI无法正确对齐人类价值观
- AI获得关键系统控制权
- AI引发军备竞赛导致灾难性错误

**2. 合成生物学**
- 设计病原体的恶意释放
- 生物安全措施失败
- 生态系统的意外破坏

**3. 核战争**
- 全面核交换导致核冬天
- 核威慑失败
- 非国家行为者的核恐怖主义

### 潜在X-Risk

**4. 气候灾难**
- 温室气体临界点触发不可逆变化
- 行星工程失败
- 海平面上升导致文明崩溃

**5. 其他自然风险**
- 大型小行星撞击
- 超级火山爆发
- 伽马射线暴

### 理论性X-Risk

**6. 虚拟世界灾难**
- 大量意识被困于痛苦模拟
- 数字存在性风险

**7. 不可知的风险**
- 未来技术带来的未知威胁

## 长期主义与TESCREAL

长期主义在TESCREAL中是"时间锚点"：

- **← 超人类主义**: 超人类主义技术可能是X-risk来源，也是X-risk解决方案
- **← 反熵主义**: 对抗熵增的终极意义在于文明的长远延续
- **← 奇点主义**: 奇点既是最大的X-risk，也是最大机遇
- **← 宇宙主义**: 成为星际物种是防范X-risk的终极保险
- **← 理性主义**: 理性评估X-risk需要概率论和决策理论
- **← 有效利他主义**: 长期主义是EA的逻辑延伸

## 批评与争议

### 主要批评方向

**可能性加权批评**
批评者质疑：我们如何确定未来会有大量人口？如果文明自我毁灭或崩溃，遥远未来的生命可能根本不存在。

**路径依赖批评**
未来不是预先决定的。当代决策不能"决定"未来——相反，未来是开放的。

**当下忽视批评**
过度关注遥远的未来可能导致对当下紧迫问题(如气候变化、贫困、不平等)的道德忽视。

**不确定性批评**
我们对未来一无所知。试图"为未来最大化"可能只是在不确定性中盲目行动。

**保守主义批评**
长期主义可能被用于为现状辩护——声称"变革会增加X-risk"。

### 哲学争议

**价值不可通约性**
功利主义框架可能无法处理不同世代之间的价值冲突。

**贴现率问题**
经济学家普遍使用正的贴现率——未来价值应被折扣。长期主义隐含地使用零或负贴现率。
